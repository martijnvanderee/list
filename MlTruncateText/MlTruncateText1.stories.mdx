# Truncate text

Truncate text, with inline styling to simulate text truncation. Note: truncating text in a `<span>` element always needs a width and/or a display property. The only way to add the text is through the default `slot`.

<Canvas>
    <Story name="Default">
        {{
            components: { MlTruncateText },
            template: `
                <ml-truncate-text :id="truncate-1" style="display:inline-block;max-width:160px;">
                    {{ text }}
                </ml-truncate-text>
            `,
            data() {
                return {
                    text: 'Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.                 };

## Truncate text with HTML

<b>TL;DR: Use span for v-html</b>

Truncate text, with inline styling to simulate text truncation. similar as above. It is important to set the slot content wrapper to `inline`, as it wouldn't show the three dots otherwise. You may use a `span` element for this.

<Canvas>
    <Story name="HTML">
        {{
            components: { MlTruncateText },
            template: `
                <ml-truncate-text :id="id" :show-button-text="showButtonText" :hide-button-text="hideButtonText" style="display:inline-block;max-width:150px;">
                    <span v-html="html" />
                </ml-truncate-text>
            `,
            data() {
                return {
                    id: 'truncate-2',
                    variant: 'collapse',
                    showButtonText: 'Lees meer',
                    hideButtonText: 'Verberg',
                    html: `Lorem ipsum <b>dolor</b> sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
                    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
                    Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur.
                    Excepteur <a href="#">sint occaecat cupidatat</a> non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.`
                };
            }
        }}
    </Story>
</Canvas>

## Truncate text collapse with HTML

In this example, a piece of truncated text can be collapsed when it has more than 2 lines in combination with the slot syntax for showing HTML or other vue components.

<Canvas>
    <Story name="CollapseHTML">
        {{
            components: { MlTruncateText },
            template: `
                <ml-truncate-text
                    :id="id"
                    :variant="variant"
                    :show-button-text="showButtonText"
                    :hide-button-text="hideButtonText"
                >
                    <div v-html="html" />
                </ml-truncate-text>
            `,
            data() {
                return {
                    id: 'truncate-2',
                    variant: 'collapse',
                    showButtonText: 'Lees meer',
                    hideButtonText: 'Verberg',
                    html: `Lorem ipsum dolor sit amet, consectetur adipiscing elit, <b>sed do eiusmod tempor</b> incididunt ut labore et dolore magna aliqua.
                            Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

                            Excepteur <a href="#">sint occaecat cupidatat</a> non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.`
